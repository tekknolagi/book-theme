Theme: Book

Installation
============

1. Download the zip file and unzip. Make sure that you have a "book" folder and not nested folders such as "book/book" (ignore the quotes).
2. Upload the unzipped folder to wp-content/themes folder.
3. Go to Appearance -> Themes and Activate the Book Theme.

Media settings
==============

Go to Settings->Media and set your media settings accordingly.

Thumbnail = 248 x 248
Medium = 450 x 0
Large = 750 x 0
